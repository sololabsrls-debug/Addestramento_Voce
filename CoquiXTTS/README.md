# Coqui XTTS v2

Modello TTS multilingua con eccellente qualità (9/10) e voice cloning zero-shot.

## Struttura Cartelle

```
CoquiXTTS/
├── scripts/          # Script di test e utilizzo
├── models/           # File del modello scaricati
├── audio_output/     # File audio generati dai test
├── config/           # File di configurazione
├── voices/           # Sample vocali di riferimento per cloning
└── README.md         # Questa documentazione
```

## Caratteristiche
- **Qualità**: 9/10 per italiano
- **Latenza**: RTF ~0.3-0.5 (buona)
- **Voice Cloning**: Zero-shot con 6-10s di audio
- **Lingue**: Multilingua (16+ lingue)
