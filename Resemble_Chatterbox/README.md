# Resemble Chatterbox

Modello TTS con miglior bilanciamento qualità/latenza (8.5/10, ~1s/frase).

## Struttura Cartelle

```
Resemble_Chatterbox/
├── scripts/          # Script di test e utilizzo
├── models/           # File del modello scaricati
├── audio_output/     # File audio generati dai test
├── config/           # File di configurazione
├── voices/           # Sample vocali di riferimento per cloning
└── README.md         # Questa documentazione
```

## Caratteristiche
- **Qualità**: 8.5/10 per italiano
- **Latenza**: ~1s per frase (eccellente)
- **Voice Cloning**: Zero-shot con 10s di audio
- **Parametri**: 500M (efficiente)
- **Ideale per**: Chiamate telefoniche real-time
